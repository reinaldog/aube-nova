**Aube Nova** is a living civilization simulator powered entirely by small language models.
12 colonists arrive on a distant world with nothing but their wits, their tools, and each other.
Every decision, every crisis response, and every newspaper headline is generated by [MiniCPM4.1-8B](https://huggingface.co/openbmb/MiniCPM4.1-8B) — in real time, ~80 tokens at a time.
You don't control them. You watch them survive.

> 🏆 *Built for the [Build Small Hackathon](https://huggingface.co/build-small-hackathon) · Track: Thousand Token Wood*

---

## 🚀 What It Does

| 🔧 System | 📖 Description |
|-----------|---------------|
| 👥 **12 LLM Agents** | Each colonist acts autonomously — their weekly action is generated by MiniCPM4.1-8B (~80 tokens in/out) |
| ⏱️ **Tick Engine** | Each week resources deplete, crises erupt, breakthroughs happen, colonists age and die |
| 📰 **The Aube Chronicle** | A newspaper front page written by the LLM at year-end, with milestone entries for deaths, births, and discoveries |
| 🔊 **Narration** | Chronicle entries read aloud by [VoxCPM2](https://huggingface.co/openbmb/VoxCPM2); gracefully falls back to gTTS on CPU |
| 📊 **Live Analytics** | Resource trends, population by generation, and cultural trait drift — all charted in real time |
| 🧬 **Cultural DNA** | Colonist personalities drift as conditions change — crises breed pragmatism, breakthroughs breed optimism |
| 🖱️ **Colonist Profiles** | Click any portrait or map token to see a colonist's traits, memories, notable actions, and health |

## 🛠️ How to Run

```bash
uv sync
cp .env.example .env   # add HF_TOKEN if using the Inference API
uv run python app.py
```

Open [http://localhost:7860](http://localhost:7860).

**Controls:**

| Button | Action |
|--------|--------|
| ▶ **ADVANCE WEEK** | Tick the simulation forward one week |
| ⏩ **FF 10 WEEKS** | Advance 10 weeks at once |
| 🗓️ **FF 1 YEAR** | Fast-forward a full colony year (52 weeks with random crises/breakthroughs) |
| 💥 **INJECT CRISIS** | Trigger a random crisis |
| 💡 **BREAKTHROUGH** | Fire an explanatory breakthrough (permanent building bonus) |
| ↺ **NEW COLONY** | Reset to a fresh colony |

## 🏗️ Architecture

```
app.py (Gradio UI)
├── simulation/   ← world state, colonists, resources, events, genetics
├── llm/          ← agent actions, chronicle, narration (VoxCPM2 + gTTS fallback)
└── ui/           ← SVG map, portraits, HUD, chronicle view, analytics charts
```

All agent decisions use **MiniCPM4.1-8B** via the Hugging Face Inference API.
Narration uses **VoxCPM2** (`openbmb/VoxCPM2`) loaded via `from_pretrained` with audio caching; gracefully falls back to gTTS on CPU-only environments.

## 💭 Philosophy

> *"Problems are inevitable. Problems are soluble."* — David Deutsch

The **Breakthrough** mechanic is inspired by *The Beginning of Infinity*: the colony doesn't just survive crises — it generates permanent explanatory knowledge that makes it more resilient. Every breakthrough is a small piece of the colony's growing understanding of its world.

The setting does narrative work for free: when oxygen is at 12% and the Reactor is critical, you don't need an engineer to explain why it matters.
